> A JAVASCRIPT LIBRARY FOR BUILDING USER INTERFACES. [来源](https://facebook.github.io/react/)

React 就是定义用户界面的 UI 库。有什么特点呢:

+ 使用可管理的小型组件构造出一个强大的应用。
+ 不需要花费时间寻找 DOM 节点，而是去维护应用的状态。
+ 代码更偏向于 声明式。
